# PartyInvitesCore
A simple application made entirely in ASP.NET Core framework, the idea of which is to create an application with an invitation to a big event for the Masters of the Universe

## Used versions of development environments
* Debian Linux/GNU 11 (bullseye) x86_64
* JetBrains Rider 2021.1
* .NET 5.0
* ASP.NET Core MVC 3
* Bootstrap 4.3.1 

### State of the project:
10.05.2021 Project is finished as all objectives are fulfilled and project is working correctly
